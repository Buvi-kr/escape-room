# 🛡️ 도기탐정: 태양풍 방어 (Dogi Detective: Solar Wind Defense)

> **포천 아트밸리 천문과학관 인터랙티브 방탈출 프로젝트**
> 
> "긴급 상황! 강력한 태양풍이 관측됐어. 지구 방어막 '오메가 베리어'를 켜야 해!"

---

## 🌌 프로젝트 개요
도기탐정 '포우리'와 함께 태양풍으로부터 지구를 구하는 웹 기반 인터랙티브 방탈출 게임입니다. 포천 아트밸리 천문과학관의 전시 데이터와 연동되어 교육적인 가치와 재미를 동시에 제공합니다.

## ✨ 주요 특징
- **몰입형 내러티브**: 전시장 동선에 맞춘 10단계 미션과 풍부한 스토리 텔링.
- **하드코어 로직**: 정답을 틀리면 처음부터 다시 시작해야 하는 긴장감 넘치는 게임 플레이 (실패 시 처음부터 재시도).
- **3단계 시나리오 시스템**: 모든 미션은 '상황 제시 - 해결 과정 - 해결 결과'의 3단계 내러티브 카드로 구성.
- **네온 퓨처리즘 디자인**: Glassmorphism과 다이나믹한 애니메이션이 적용된 프리미엄 UI.
- **지속성 보장**: 브라우저 `localStorage`를 사용하여 실시간 세이브/로드 시스템 구현.

## 🛠 기술 스택
- **Core**: Vanilla JavaScript (ES6+), HTML5
- **Styling**: CSS3 (Glassmorphism, Neon Futurism Design)
- **Asset**: AI-Generated & WebP optimized images
- **Architecture**: Single Page Application (SPA)

## 🧩 미션 리스트
1.  **시스템 로그인**: ISS 궤도 데이터 동기화
2.  **산소 시스템**: 지구 탄생 데이터 분석
3.  **코어 베리어**: 외핵 에너지 활성화
4.  **중력 동기화**: 테이아 충돌 데이터 분석
5.  **좌표 매핑**: 궁수자리 좌표 확보
6.  **시야 보정**: 케플러식 망원경 초점 조정
7.  **회전 동력**: 목성 자전 동력 연결
8.  **이동 경로**: 월면차 이동 경로 확보
9.  **시스템 안정**: 히파르코스 별의 밝기 데이터 적용
10. **베리어 가동**: 주계열성 에너지 최종 가동

## 🚀 시작하기
1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다. (별도의 서버 설정 없이 바로 실행 가능합니다.)
3. 포우리와 함께 지구를 지키세요!
