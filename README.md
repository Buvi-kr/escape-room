# 🛡️ 도기탐정: 태양풍 방어 (Dogi Detective: Solar Wind Defense)

> **포천 아트밸리 천문과학관 인터랙티브 방탈출 프로젝트**
> 
> "긴급 상황! 강력한 태양풍이 관측됐어. 지구 방어막 '오메가 베리어'를 켜야 해!"

---

## 🌌 프로젝트 개요
도기탐정 '포우리'와 함께 태양풍으로부터 지구를 구하는 웹 기반 인터랙티브 방탈출 게임입니다. 포천 아트밸리 천문과학관의 전시 데이터와 연동되어 교육적인 가치와 재미를 동시에 제공합니다.

## ✨ 주요 특징
- **하이브리드 분할 레이아웃**: 상단 모니터 박스(비주얼)와 하단 컨트롤 영역(상호작용)을 물리적으로 분리하여 가독성과 조작성 극대화.
- **실시간 시스템 HUD**: 미션 에셋(이미지)이 없는 상황에서도 현재 스테이지 상태와 에너지 충전율을 시각화하는 지능형 모니터링 시스템.
- **몰입형 내러티브**: 전시장 동선에 맞춘 10단계 미션과 '상황 제시 - 해결 과정 - 해결 결과'의 3단계 스토리텔링 구조.
- **하드코어 로직**: 보안 규정에 따라 오답 시 즉시 데이터가 소거되어 처음부터 다시 시작 (긴장감 넘치는 게임 플레이).
- **네온 퓨처리즘 디자인**: Glassmorphism, CRT 스캔라인 효과, 다이나믹 스테이지 글로우가 적용된 프리미엄 우주 SF 스타일.
- **통합 가이드 시스템**: 별도의 힌트 팝업 없이 문제 아래에 전시관 동선을 안내하는 실시간 가이드 제공.

## 🛠 기술 스택
- **Core**: Vanilla JavaScript (ES6+), HTML5
- **Styling**: CSS3 (Custom Properties, Glassmorphism, Mobile-First Hybrid Layout)
- **Visuals**: AI-Generated WebP Assets & Canvas Starfield Engine
- **Storage**: Browser `localStorage` 기반 실시간 상태 보존 시스템

## 🧩 미션 리스트
1.  **시스템 로그인**: ISS 궤도 데이터 동기화
2.  **산소 시스템**: 지구 탄생 데이터 분석 (바다)
3.  **코어 베리어**: 지구 내부 구조 분석 (외핵)
4.  **중력 동기화**: 테이아 충돌 데이터 분석 (달 탄생)
5.  **좌표 매핑**: 황도 12궁 궁수자리 좌표 확보
6.  **시야 보정**: 케플러식 망원경 광학 교정
7.  **회전 동력**: 목성 자전 주기 동력 연결
8.  **이동 경로**: 월면차(LRV) 이동 경로 확보
9.  **에너지원 식별**: 항성(Star)의 물리적 정체 파악
10. **베리어 가동**: SOHO 태양 관측 위성 데이터 동기화

## 🚀 시작하기
1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다. (모바일 뷰포트에 최적화되어 있습니다.)
3. 도기 탐정 '포우리'와 함께 10개의 지점을 탐험하며 지구를 지키세요!
