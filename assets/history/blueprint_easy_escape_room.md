## 3. 스토리라인 — 태양풍 방어 대작전

### 📖 배경 설정
> 🚨 **긴급 통신!** 강력한 태양풍이 지구를 향해 접근 중!
> 포천 아트밸리 천문과학관에 설치된 **'오메가 방어 시스템'**을 가동해야 합니다.
> 하지만 시스템이 잠겨있고, 핵심 코드가 과학관 전시실 곳곳에 분산되어 있어요!
> 🐕 **도기 탐정**이 9개의 미션을 해결하고 방어 시스템을 완전히 가동시켜 주세요!

### 흐름 요약
```
[타이틀] → [프롤로그: 긴급통신]
  → [1전시실: 미션1~3] 시스템 접근 + 산소 복구 + 코어 잠금해제
  → [2전시실: 미션4~6] 에너지킷 해제 + 방어막 정화 + 동력 충전
  → [휴게실+3전시실: 미션7~9] 좌표 확보 + 시스템 안정화 + 베리어 가동
  → [엔딩: 태양풍 방어 성공 + 결과 화면 + 상품 안내]
```

### 미션별 서사 흐름

| 미션 | 서사 맥락 (도기탐정 대사) | 시스템 행동 |
|------|----------------------|-----------|
| M1 | "시스템에 접근하려면 비밀번호가 필요해!" | 🔐 방어 시스템 로그인 |
| M2 | "산소 시스템이 고장났어! 복구해야 해!" | 🫁 산소 공급 복구 |
| M3 | "에너지 코어가 잠겼어! 해제 코드를 찾자!" | 🔓 코어 잠금 해제 |
| M4 | "방어 에너지 키트를 활성화해야 해!" | 🔋 에너지킷 해제 |
| M5 | "방어막에 오류가 있어! 정화 작업이 필요해!" | 🧹 방어막 정화 |
| M6 | "동력이 부족해! 충전 코드를 입력하자!" | ⚡ 동력 충전 |

### 7-3. localStorage 저장/복원
```javascript
const SAVE_KEY = "dogi_solar_defense";

function saveGame() {
  const data = {
    nickname: state.nickname,
    currentQuiz: state.currentQuiz,
    startTime: state.startTime,    // 게임 시작 시 Date.now()
    hintsUsed: state.hintsUsed,
    completed: state.completed     // [0,1,2,...] 완료 문제 인덱스
  };
  localStorage.setItem(SAVE_KEY, JSON.stringify(data));
}

function loadGame() {
  const raw = localStorage.getItem(SAVE_KEY);
  if (raw) return JSON.parse(raw);
  return null;
}

// 결과 화면용
function getElapsedTime() {
  const elapsed = Date.now() - state.startTime;
  const min = Math.floor(elapsed / 60000);
  const sec = Math.floor((elapsed % 60000) / 1000);
  return `${min}분 ${sec}초`;
}
```
